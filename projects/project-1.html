Conway's Game of Life

Made with WxWidgets
Created 4/2024

As part of the Project and Portfolio I course at Full Sail University, I developed an application that simulates Conway's Game of Life using the WxWidgets API. This was a one month long solo project, and it served as an assessment of our skills in object-oriented programming, file I/O, and algorithm implementation using C++. was my first major project at Full Sail, and I feel it provided a strong test of my programming skills and my ability to work with an API that I was unfamiliar with.
