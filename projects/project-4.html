Project Meltdown

Made with Unreal Engine Created 4/2025

Project Meltdown is a round based multiplayer horde like shooter where the players work together to defeat hordes of automatons using a variety of weapons and cybernetic enhancements.

Project Meltdown was started the same month as Echo in the Shadows, however I did not join the team working on this game until the following month, when we entered Open Portfolio. The most notable feature that I have implemented would have to be multiplayer, making use of Steam's Online Subsystem. This project has been an amazing test of my skills that I have gained during my time at Full Sail University, and I plan to continue working on it even after I graduate. You can find Project Meltdown here: https://quantexgames.itch.io/project-meltdown
